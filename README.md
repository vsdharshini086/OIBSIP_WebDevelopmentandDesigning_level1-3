# Temperature Converter

1. Objective
To create a simple temperature converter that converts values between Celsius, Fahrenheit, and Kelvin.

2. Steps Performed
- Designed the user interface for input and output  
- Implemented conversion logic using JavaScript  
- Added validation for user input  
- Tested conversions for accuracy  

3. Tools Used
- HTML  
- CSS  
- JavaScript  

4. Outcome
A user-friendly temperature converter that provides accurate results instantly.
